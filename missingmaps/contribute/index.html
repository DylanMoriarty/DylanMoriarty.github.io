Beginners

1.

Create an Open Street Map account

Go to the OpenStreetMap website and register to create an account. Follow the link in the email they send to confirm your account.

2.

Find a task to map

Go to the Task Manager to find a project and task that interests you.

Learn how to lock a task to edit, select an editing program and change the background settings.

iD editor is recommended for beginners.

3.

Learn to map buildings and roads

Once you've found a task, learn to map building and roads.

Details on OpenStreetMap's tagging system can be found on the map features wiki.

4.

Save and complete your task

It's important to save regularly. If you finish the task mark it as complete; however, if it's not complete unlock it for others to complete. Remember, you may leave a message for the next mapper.
