/ Election Worker

Government websites aren’t the best things.Unless they’re 18F. Those guys do great stuff.

They contain invaluable resources for how to get involved with government, but lack the uniformity and densly packed navigation makes accessing those resources difficult. It feels like working through a virtual filing cabinet where the folder labels are only vague suggestions of what they might contain.

One such resource is how you can work election polls. WorkElections is a tool DevelopmentSeed built with the Fair Elections Network & Election Academy to make it easier for folks to find that out!

Through the site you can enter your state, county, city, zip code or address and be taken to details on what the requirements are to work a polling station and direct links to where you can sign up to get started!

The database is manually updated by FELN, and is grabbable by an APIYou can check out the API here.. As it’s manually updated, there’s only a handfulAZ, CA, FL, NV, NM, OH, VA of states currently covered. More are to be added down the line.
